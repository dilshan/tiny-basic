# TO

**Category:** Statement / Keyword Modifier  
**Syntax:** `FOR variable = start_expr TO limit_expr`

## Description

Specifies the terminal boundary condition in a `FOR` loop statement.

## See also

- [FOR](for.md)
- [STEP](step.md)

## Example

```basic
10 FOR A = 1 TO 10
20 PRINT(A)
30 NEXT A
```
