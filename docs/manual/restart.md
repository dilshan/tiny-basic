# RESTART

**Category:** Keyword / Command Modifier  
**Syntax:** `I2C(RESTART)`

## Description

Command argument for `I2C` transactions to perform a repeated start condition without releasing the bus control.

## See also

- [I2C](i2c.md)
- [START](start.md)
- [READ](read.md)
- [WRITE](write.md)

## Example

```basic
10 I2C(RESTART)
```
