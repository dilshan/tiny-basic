# STOP

**Category:** Keyword / Command Modifier  
**Syntax:** `I2C(STOP)`
`SPI(STOP)`

## Description

Used as a command argument within `I2C` or `SPI` to release the communication select lines or issue I2C stops.

## See also

- [I2C](i2c.md)
- [SPI](spi.md)
- [START](start.md)
- [RESTART](restart.md)

## Example

```basic
10 I2C(STOP)
```
