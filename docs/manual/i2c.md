# I2C

**Category:** Function  
**Syntax:** `I2C(INIT)`
`I2C(START)`
`I2C(RESTART)`
`I2C(STOP)`
`I2C(WRITE, num)`
`I2C(READ, ack)`

## Description

Controls Inter-Integrated Circuit (I2C) communication bus peripherals. Supports commands for initializing the port, transmitting start/restart/stop conditions, writing bytes, or reading bytes.
