# THEN

**Category:** Statement / Keyword Modifier  
**Syntax:** `IF boolean_expr THEN statement`
`IF boolean_expr THEN`

## Description

Used in conjunction with the `IF` statement to delineate the boundary between the condition and the action to be performed if true.

## See also

- [IF](if.md)
- [ELSE](else.md)

## Example

```basic
10 IF A = 1 THEN GOTO 100
```
