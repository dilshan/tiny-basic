# OCT

**Category:** Print Format Modifier  
**Syntax:** `PRINT(OCT, expression_list)`

## Description

Used as a prefix inside a `PRINT` statement to format output representation of numbers in Octal format.

## See also

- [HEX](hex.md)
- [BIN](bin.md)

## Example

```basic
10 LET A = 8
20 PRINT(OCT, A)
```
