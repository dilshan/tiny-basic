# XNOR

**Category:** Bitwise Operator  
**Syntax:** `expr1 XNOR expr2`

## Description

Performs a bitwise XNOR operation on the two integer expressions. Returns true where bits in both operands match.

## See also

- [XOR](xor.md)
- [EQV](eqv.md)

## Example

```basic
10 LET A = 12 XNOR 10
20 PRINT(A)
```
