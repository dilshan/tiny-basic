# BITGET

**Category:** Function  
**Syntax:** `BITGET(num, num_pos)`

## Description

Queries and returns the state (0 or 1) of the bit at the 0-indexed position specified by `num_pos`.

## See also

- [BITSET](bitset.md)
- [BITCLR](bitclr.md)

## Example

```basic
10 LET A = BITGET(9, 3) 
20 PRINT(A)
```
