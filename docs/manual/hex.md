# HEX

**Category:** Print Format Modifier  
**Syntax:** `PRINT(HEX, expression_list)`

## Description

Used as a prefix inside a `PRINT` statement to format output representation of numbers in Hexadecimal format.

## See also

- [BIN](bin.md)
- [OCT](oct.md)

## Example

```basic
10 LET A = 255
20 PRINT(HEX, A)
```
