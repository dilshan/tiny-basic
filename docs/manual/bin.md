# BIN

**Category:** Print Format Modifier  
**Syntax:** `PRINT(BIN, expression_list)`

## Description

Used as a prefix inside a `PRINT` statement to format output representation of numbers in raw Binary format.

## See also

- [HEX](hex.md)
- [OCT](oct.md)

## Example

```basic
10 LET A = 5
20 PRINT(BIN, A)
```
