# START

**Category:** Keyword / Command Modifier  
**Syntax:** `I2C(START)`
`SPI(START)`

## Description

Used within `I2C` or `SPI` functions to acquire communication control (SPI chip select or I2C start).

## See also

- [I2C](i2c.md)
- [SPI](spi.md)
- [STOP](stop.md)
- [RESTART](restart.md)

## Example

```basic
10 SPI(START)
```
